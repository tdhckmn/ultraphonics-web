CSS Variables

Use these CSS custom properties for consistent styling across the website.

Color Variables

:root {
  /* Primary Colors */
  --color-primary-blue: #008c8d;
  --color-primary-red: #f60403;
  --color-accent-cyan: #00ddde;
  --color-accent-hover: #33ddff;

  /* Button-specific */
  --color-btn-bg: #008c8d;
  --color-btn-hover: #00aeb0;

  /* Backgrounds */
  --color-background: #1a1a1a;
  --color-background-dark: #121212;

  /* Text Colors */
  --color-text: #ffffff;
  --color-text-secondary: #cccccc;
  --color-text-tertiary: #aaaaaa;
  --color-white: #ffffff;
}

Shadow Variables

:root {
  /* Glow effect - signature cyan glow */
  --shadow-glow: 0 0 10px rgba(0, 221, 222, 0.5),
                 0 0 20px rgba(0, 221, 222, 0.3),
                 0 0 30px rgba(0, 221, 222, 0.2);

  /* Standard shadows */
  --shadow-sm: 0 2px 4px rgba(0, 0, 0, 0.2);
  --shadow-md: 0 4px 8px rgba(0, 0, 0, 0.3);
}

Typography

Font Import

Add this to the top of your CSS file or in the <head> of your HTML:

@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap');

Font Stack

body {
  font-family: 'Montserrat', -apple-system, BlinkMacSystemFont, 'Segoe UI', Arial, sans-serif;
}

  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

Secondary CTA / Outline Button

Usage: Use .button-outline or .button.button-outline for secondary CTAs (e.g., "Request a Song", "Download Wedding Flyer"). This provides a transparent button with colored border that fills on hover.

.button-outline {
  background-color: transparent;
  border: 2px solid var(--color-btn-bg);
  color: var(--color-text);
  padding: 12px 24px;
  font-size: var(--font-size-base);
  border-radius: 5px;
  text-decoration: none;
  transition: all 0.3s ease;
  cursor: pointer;
  display: inline-block;
  font-weight: bold;
}

.button-outline:hover {
  background-color: var(--color-btn-bg);
  color: var(--color-white);
}

/* HTML Example */
<a href="link.html" class="button button-outline">Secondary Action</a>

CTA Button Guidelines

  • Primary CTA: Use .button for main actions (e.g., "Book Us", "Get a Quote", "Request a Quote"). Filled background with hover lift effect.
  • Secondary CTA: Use .button-outline for supporting actions (e.g., "Request a Song", "Download Wedding Flyer", "View Services"). Transparent with border that fills on hover.
  • When asked to create a new CTA: Clarify if it should be primary (filled) or secondary (outline) based on its importance in the user journey.

Accessibility Considerations

Color Contrast

  • White text on dark backgrounds: Meets WCAG AAA standards
  • Cyan (#00ddde) on dark backgrounds: Meets WCAG AA standards
  • Blue (#008c8d) buttons with white text: Meets WCAG AAA standards

Best Practices

  • Always test color combinations for sufficient contrast
  • Don't rely on color alone to convey information
  • Ensure focus states are visible (use cyan glow effect)
  • Maintain minimum font size of 0.9rem (14.4px)
  • Use semantic HTML elements
  • Provide alt text for all images

Focus States

button:focus,
a:focus,
input:focus {
  outline: none;
  border-color: var(--color-accent-cyan);
  box-shadow: var(--shadow-glow);
}
