Typography Guidelines

Headings

  • Use bold (700) or semi-bold (600) weights
  • Section headings in bright cyan (#00ddde)
  • Add letter-spacing for emphasis (1-2px)
  • Use text-transform: uppercase sparingly

Body Text

  • Regular (400) weight for body copy
  • Line-height: 1.6 for readability
  • White (#ffffff) for primary text
  • Gray (#cccccc) for secondary text

Interactive Text

  • Bold (700) weight for buttons
  • Cyan (#00ddde) for links
  • Underline on hover for links
  • Uppercase for CTAs (with letter-spacing)

Chromatic Aberration Effect

Our signature visual effect uses red/cyan offset to create a dynamic, energetic aesthetic reminiscent of 3D glasses and vintage print techniques.

The Effect

The chromatic aberration creates a sense of motion and energy. It's most prominently featured in our logo where the red and cyan create a spiral effect with a slight offset.

Usage

  • Primary use: Logo design
  • Secondary use: Special graphics and promotional materials
  • Avoid: Body text, small text, or excessive use that reduces readability

Implementation

When creating graphics with this effect:

  • Red layer offset slightly to one side
  • Cyan layer offset to the opposite side
  • White/black central layer for definition
  • Keep offset subtle (2-5px for most uses)
